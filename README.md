# Hasan AI Research

Public GitHub Pages homepage for `https://HasanAIResearch.github.io/`.

## Publish

Create a public GitHub repository named `HasanAIResearch.github.io`, then push this folder to `main`.

GitHub Pages should publish automatically for a user-site repository with this exact name. If not, enable:

Settings -> Pages -> Deploy from branch -> main / root

## Content Rules

- Keep pages public-safe.
- Do not include private portfolio/account details.
- Do not include HH/internal scanner outputs unless rewritten as public methodology.
- Add sources and freshness notes on research pages.
